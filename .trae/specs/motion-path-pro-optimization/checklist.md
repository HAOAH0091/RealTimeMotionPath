# Checklist

- [x] UI Panel: A "Style Settings" panel is available in the Motion Path Pro tab.
- [x] UI Panel: Path width, path color, frame point size, frame point color, keyframe point size, keyframe point color (normal/selected), handle line width, handle line color (normal/selected), handle endpoint size, handle endpoint color (normal/selected) are customizable.
- [x] Path Drawing: Motion path line color and width update immediately when settings change.
- [x] Frame Points: Small circular points are drawn on the path for every frame.
- [x] Frame Points: Points update size and color when settings change.
- [x] Frame Points: Points can be toggled on/off.
- [x] Keyframe Points: Keyframe points use the custom size and color settings.
- [x] Keyframe Points: Selected keyframes use the selected color.
- [x] Handles: Handle endpoints are circular.
- [x] Handles: Handle lines use the custom width and color settings.
- [x] Handles: Handle endpoints use the custom size and color settings.
- [x] Handles: Selected handles use the selected color.
