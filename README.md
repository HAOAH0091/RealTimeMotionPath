# RealTimeMotionPath - 实时运动路径插件

## 简介

RealTimeMotionPath 是一个 Blender 动画插件，可以在 3D 视口中实时显示和编辑物体或骨骼的运动路径。你可以直接在 3D 视图中拖拽关键帧、调整曲线手柄，让动画制作更加直观和高效！
目前只兼容blender 5.0以上版本，其他版本并没有进行测试，无法保证稳定性。

## 主要功能

- 🎯 **实时运动路径显示** - 在 3D 视口中直接看到物体/骨骼的运动轨迹
- ✋ **直接手柄编辑** - 拖拽贝塞尔手柄调整曲线，和在图形编辑器里一样方便
- 🔄 **多对象/多骨骼同时显示** - 可以同时看到多个选中对象或骨骼的运动路径
- 🎨 **自定义样式** - 可以调整路径颜色、粗细、点大小等外观

## 快速开始

### 1. 启用插件

在 3D 视图顶部的标题栏中，找到并点击这个图标：🎯 (IPO_BEZIER)

点击后运动路径就会开始显示！

<img width="343" height="48" alt="image" src="https://github.com/user-attachments/assets/836f0a5c-52f6-44b6-9f45-24de76753c16" />

### 2. 显示运动路径

- **物体模式**：选中一个或多个有动画的物体，运动路径会自动显示
- **姿态模式**：选中一个或多个有动画的骨骼，它们的运动路径会显示出来

### 3. 编辑关键帧

1. 点击路径上的黄色圆点（关键帧点）选中它
2. 按住 Shift 或 Ctrl 可以多选多个关键帧
3. 拖拽选中的关键帧可以改变位置
4. 点击空白区域可以取消选中

### 4. 调整曲线手柄

1. 点击关键帧两侧的手柄线
2. 拖拽手柄端点可以调整曲线形状
3. 右键点击关键帧可以选择手柄类型（自由、对齐、矢量、自动等）

## 界面说明

### 顶部标题栏菜单

点击标题栏中的 🎯 图标旁边的小箭头，可以打开设置面板：

- **更新模式**
  - **智能 (事件)** - 推荐！只在关键帧数据变化时更新，最省性能
  - **计时器 (轮询)** - 定期检查更新，更稳定但性能消耗较高

- **交互 FPS** - 限制拖拽操作的帧率，节省性能

- **自动更新 FPS** (仅计时器模式) - 设置自动检查的频率

### 插件首选项设置

在 Blender 偏好设置 › 插件 › RealTimeMotionPath 中，可以调整外观样式：

- **路径样式** - 调整路径线的粗细和颜色
- **帧点** - 显示/隐藏普通帧的小点，调整大小和颜色
- **关键帧** - 调整关键帧点的大小和颜色（选中/未选中）
- **手柄** - 调整手柄线和端点的样式
- **原点指示器** - 调整对象原点显示的样式（圆环、圆点、或圆环+圆点）

## 使用技巧

1. **多选操作** - 按住 Shift 或 Ctrl 选中多个关键帧后，可以一起拖拽移动，不需要一直按住按键

2. **批量设置手柄** - 多选关键帧后右键，可以一次性设置所有选中关键帧的手柄类型

3. **父子级对象** - 插件会自动处理有父级的对象或子骨骼，路径会显示在正确的位置

4. **交互时的路径** - 当你移动物体时（按 G 键），路径会暂时保持不变，等你松手后自动更新

## 常见问题

**Q: 为什么我选中对象后看不到运动路径？**

A: 确保对象有动画关键帧，并且你已经点击了标题栏中的 🎯 图标启用了插件。

**Q: 可以同时显示多个对象的路径吗？**

A: 可以！同时选中多个对象，它们的运动路径都会显示出来，你可以对任意一个进行编辑。

**Q: 插件会影响 Blender 的性能吗？**

A: 默认使用「智能模式」，只会在关键帧数据变化时更新，对性能影响很小。如果遇到卡顿，可以尝试降低「交互 FPS」设置。
